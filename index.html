The Scot Bot

Senior Project

If you have ever taken a college course… you know how important syllabi are to your success in that course. There is often a lot of information about the course that is necessary to know. One problem that always seems to resurface is disorganization. This usually results in lost syllabi. So, a simple solution is to put the syllabi all in one spot. However, there is still a inconvenience factor with syllabi often ranging from 1-10 pages worth of information. The Scot Bot’s purpose is to diminish syllabi overload and alleviate the inconvenience of searching through pages of information. The Scot Bot is a bot that will answer questions that students have about syllabi. This bot uses AI technology, specifically a variant of ChatGPT, to answer questions from a dataset of syllabi. The Scot Bot utilizes a combination of Zero-Shot Learning and a Limited Keyword method to limit the dataset and provide concise and accurate responses. A feat of The Scot Bot is its generality. It can be implemented with ease for other universities and colleges.

Language: Python
Focus: AI, Machine Learning, Data Cleaning
Aug 2022 - May 2023, Monmouth College

Ostep Projects - File Systems Checker

Operating Systems

The purpose of this project was to create a file system checker that would verify the integrity of a file system image. Using C, I developed an algorithm that parsed through the superblock, inodes, bitmaps, and data blocks to ensure that the file system was consistent. This project gave me a deep understanding of how files are stored and tracked in memory. A key component of this project was the use of the mmap() function. This function allows a program to map a file into memory, which makes it possible to access the file's data directly. Using mmap(), I was able to efficiently parse through the file system image and check for errors. Overall, this project expanded my understanding of the operating system and increased my ability to write clean and efficient C code. I am now better equipped to develop software that interacts with the file system.
