Monocipher

Project Introduction

This project was for my Data structures and Algorithms class. In this project, we created a command-line program using python. This program can encrypt and decrypt text with a simple alphabet key. Key generation is also a feature that is offered. Key generation can be done with a regular alphabet and shifted/reversed or the user can insert a keyword before shifting and reversing. The final feature that I implemented in this program was the ability to do single and multi-letter frequency analyses. The English language uses certain three letter, two letter, and single-letter words more often than other words. Frequency analysis can help the user figure out where they might be placed.

Usuage

*****************MonoCipher*****************
Decipher(-d):
Command Line Usage: python3 monocipher.py -d cipher.txt key.txt plain.txt
-d will Decipher an encrypted text with a given key and then output a file with the decrypted text

Encryption(-e):
Command Line Usage: python3 monocipher.py -e plain.txt key.txt cipher.txt
-e will Encrypt text with a given key and output the excrypted text file.

Key Generation(-k):
Command Line Usage: python3 monocipher.py -k key.txt -r
Command Line Usage: python3 monocipher.py -k key.txt -s
Command Line Usage: python3 monocipher.py -k key.txt -s 4
Command Line Usage: python3 monocipher.py -k key.txt -r -s 4
Command Line Usage: python3 monocipher.py -k key.txt -r -s -4
-k will generate a key for encryption. using -k then follow by a -r will create a reversed key. -k followed by -s will shift the key to the right by a specified number if there is not a specified number then it will shift 1 by default. If you want to shift left the input needs to be positive if you want to shift right the input after -s needs to be negative.

Key Generation With Keyword(-k), (-kw):
Command Line Usage: python3 monocipher.py -k key.txt -kw "keyword" -r
Command Line Usage: python3 monocipher.py -k key.txt -kw "keyword" -s
Command Line Usage: python3 monocipher.py -k key.txt -kw "keyword" -s 4
Command Line Usage: python3 monocipher.py -k key.txt -kw "keyword" -r -s 4
Command Line Usage: python3 monocipher.py -k key.txt -kw "keyword" -r -s -4
-k will generate a key for encryption. -kw will then insert a keyword at the beginning of the key. using -k then follow by a -r will create a reversed key. -k followed by -s will shift the key to the right by a specified number if there is not a specified number then it will shift 1 by default. If you want to shift left the input needs to be positive if you want to shift right the input after -s needs to be negative.

Frequency Analysis(-a):
Command Line Usage: python3 monocipher.py -a cipher.txt Analysis.txt
-a will do a single letter frequency analysis on the cipher text and write it to the second text file that was specified.

Multiple Letter Frequency Analysis(-a):
Command Line Usage: python3 monocipher.py -a 2 cipher.txt Analysis.txt
Command Line Usage: python3 monocipher.py -a 3 cipher.txt Analysis.txt
-a followed by another number will do a multiple letter frequency analysis on the cipher text and write it to the second text file that was specified.
